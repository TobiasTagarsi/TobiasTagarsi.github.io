Caretaker

Final Project

Techniques and Skills:
-C# Unity
-Builders and Factories
-Pathfinding AI

A Project that was inspired by the game Startopia. It using Unity's built in AI pathfinding tools to have sentient shapes walk about a space station, while the user creates facilities to fulfil the needs of each alien. The alien's needs are slowly depleting bars such as hunger. To facilitate some needs, the user has to also hire aliens to work at the facilities. The user can place down facilities where they want, and the pathfinding system will guide the aliens to the correct locations. The user has a choice of different sized facilities. The project was built to showcase my ability to use Object Orientation Programming techniques and design patterns such as builders and factories.

Rocket Racer

Solo Unity Project

Techniques and Skills:
-C# Unity
-Custom AI

A simple racing game using simple AI. Player choose map and difficulty, racing against AI opponents who pathfind using checkpoints and random noise (data variation). Player controls racing rocket with WASD and spacebar (handbreak). AI have different speeds on different difficulties.

Space Buttons

Solo Unity Virtual Reality Project

Techniques and Skills:
-C# Unity
-Virtual Reality
-Collision physics

A timed button pushing game that implements third-party (not made by be) handtracking software with the HTC Vive VR kit. The third-party software tracks the players real hands with the VR kit which I then used to create buttons to be pushed in regualr intervals. Failing to push the button results in failure. The challenge with this project was getting the buttons to work correctly with the hand-tracking collision physics.

Tower of London Information Application

Group History Info App

Techniques and Skills:
-C# Unity
-Augmented Reality
-Teamwork
-Production Cycles

The AR software itself is third-party. Group work: I was the programmer in a team of 3 user experience designers (UX) and 1 game designer. I worked with the game designer to produce an prototype artefact based on the UX's work, as well as implement the game designers' game. There are two sections, a free-roam AR game and a target based AR information application. The target info-app displays models of the tower of london and provide historical information using a phone's camera and target image. The game section allows the user to select the ground near them with a phone's camera, which then creates a world around them to explore by facing a direction and tapping the screen. By interating with non-controllable chacters, the player obtains hints of locations to visit, providing both a means of direction and a platform to teach and show historical informaiton/sites.
